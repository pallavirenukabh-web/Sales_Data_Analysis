Sales Data Analysis Project
Tools Used
* MySQL
* Power BI

Project Overview
This project analyzes sales data to identify key insights such as top-performing cities, regions, and customer behavior.

Key Insights
* Los Angeles generates the highest sales, indicating strong market demand and future business potential.
* Top 5 cities contribute significantly to overall revenue.
* Sales performance varies across regions and product categories.
